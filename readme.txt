Verify your Gumroad's customers licenses right within WordPress.

== Description ==

Are you a seller of licensing digital products on Gumroad? And do you need to verify the licenses of your customers when they request technical support from you or something else?
This plugin was specially developed for this matter as it relies on the Gumroad API and is easy to install and use.

= How it works =
STEP 1:
1. Create a [Gumroad](https://gumroad.com/settings/advanced#application-form "Gumroad App") on your profile settings.
2. Click on Generate your access token.
3. Copy/Paste the generated access token on the plugin API Authentication field.
4. Save changes!

STEP 2:
1. Copy/Paste this shortcode `[vclg_form]` on whatever place you want the verify licenses form to show up.

That's it!! Well done.

== Installation ==

= 1. ADMIN SEARCH =
1. In your Admin, go to menu Plugins > Add.
2. Search for "Verify Customers Licenses for Gumroad".
3. Click to install.
4. Activate the plugin.
5. A new menu item "VCLG Options" will appear under your Comments menu option.

= 2. DOWNLOAD & UPLOAD =
1. Download the plugin (a zip file) on the right column of this page.
2. In your Admin, go to menu Plugins > Add.
3. Select the tab “Upload”.
4. Upload the .zip file you just downloaded.
5. Activate the plugin.
6. A new menu item "VCLG Options" will appear under your Comments menu option.

= 3. FTP UPLOAD =
1. Download the plugin (.zip file) on the right column of this page.
2. Unzip the zip file contents.
3. Upload the gumroad folder to the /wp-content/plugins/ directory of your site.
4. Activate the plugin through the ‘Plugins’ menu in WordPress.
5. A new menu item "VCLG Options" will appear under your Comments menu option.

== Frequently Asked Questions ==

= Where can I find my access token? =

By creating a  [Gumroad](https://gumroad.com/settings/advanced#application-form "Gumroad App") on your profile settings click on generate "access token".

= Is this add-on safe and won't cause me problems with my site? =

Yes, it is safe, and we used Gumroad API that only you can access after giving the permissions through your generated access code.
Also, the extension appears only after an administrator has logged in.

= I installed this add-on, but it doesn't work as intended! =

Then do not hesitate to open a ticket in the forums of the plugin, and we will try to solve the problem ASAP!
